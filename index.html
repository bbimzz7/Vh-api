SERIUS.

VertictHub adalah platform distribusi script Roblox yang dikelola secara independen. Setiap script dikembangkan secara manual, diuji secara menyeluruh, dan diperbarui sesuai kebutuhan pengguna.

Sistem key kami memastikan bahwa setiap pengguna terautentikasi dengan tepat — satu key terikat ke satu perangkat dan satu username Roblox. Tidak dapat dibagikan. Tidak dapat disalahgunakan.

Untuk pertanyaan, laporan kesalahan, atau informasi pembaruan, komunitas kami aktif di Discord dan TikTok.

Key Terverifikasi

HWID dan username Roblox terikat ke setiap key. Tidak dapat digunakan di perangkat atau akun yang berbeda.

Pembaruan Berkala

Script diperbarui mengikuti perubahan game. Pengguna memperoleh versi terbaru tanpa perlu mengunduh ulang.

Komunitas Aktif

Server Discord tersedia untuk dukungan teknis, diskusi, dan informasi rilis script terbaru.

Sepenuhnya Gratis

Seluruh layanan tersedia tanpa biaya. Key dapat diambil kembali setiap 24 jam secara gratis.

Cara Pemasangan

EMPAT
LANGKAH.

Proses dari pengambilan key hingga script berjalan membutuhkan waktu kurang dari lima menit. Ikuti setiap langkah secara berurutan.

01

Bergabung ke Discord

Semua informasi script, tautan unduhan, dan pembaruan hanya dibagikan melalui server Discord resmi. Keanggotaan bersifat wajib untuk mendapatkan akses ke file script.

Bergabung ke Discord
02

Ambil Key

Kunjungi halaman Ambil Key. Tersedia dua metode pengambilan:

Melalui browser — masukkan username Roblox secara manual. HWID browser terdeteksi otomatis dan terikat ke key yang diterbitkan.

Melalui script — klik tombol di dalam popup script. Browser membuka tautan berisi data terenkripsi. Key diterbitkan secara otomatis. Key berlaku selama 24 jam.

Ke Halaman Ambil Key
03

Unduh Script dari Discord

Setelah key diterima, unduh file .lua dari channel yang tersedia di Discord. Simpan file di folder yang dikenali executor — biasanya workspace atau autoexec.

Script hanya tersedia melalui server Discord resmi. File dari sumber lain tidak dapat dijamin keamanannya.
04

Jalankan di Executor

Buka game Roblox yang didukung, hubungkan executor, lalu jalankan script. Apabila muncul jendela verifikasi key, tempelkan key yang telah diperoleh.

Username Roblox harus identik dengan username yang digunakan saat pengambilan key. Perbedaan huruf kapital sekalipun akan menyebabkan verifikasi gagal. Key juga terikat ke satu perangkat dan tidak dapat digunakan secara bersamaan di perangkat lain.
